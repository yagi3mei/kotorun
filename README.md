# ことラン（KotoRun）

日本語学習用の練習ゲームサイトです。

ひらがな・カタカナ・単語・助数詞などを、
ゲーム感覚で繰り返し練習できるように作成しています。

## サイトURL

https://（https://yagi3mei.github.io/kotorun/index.html）

---

## 主な機能

- かなゲーム
- かるたゲーム
- 助数詞ゲーム
- 音声読み上げ
- タイム計測
- ミス数表示

---

## 開発環境

- HTML
- CSS
- JavaScript

---

## 注意事項

音声読み上げは、
使用するブラウザや端末によって
発音が異なる場合があります。
